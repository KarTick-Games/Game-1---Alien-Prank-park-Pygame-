What I Built
Core Features
🔓
Progressive Unlock System Levels unlock sequentially; state persists via JSON save file between sessions
⚙️
Developer Mode Toggle Single flag unlocks all levels instantly — designed for rapid iteration and testing
📐
Resolution-Independent Rendering Virtual canvas scales to any window size with letterboxing and coordinate remapping
🎭
9-Costume Character System Palette-driven draw_player() renders every outfit from a single unified function
🌌
Animated Space Background 120 procedurally placed stars scroll downward and loop, creating depth parallax
🗂️
Modular Level Architecture Each level lives in its own file (Level1.py … Level10.py), imported dynamically
🌍
5-Act World Map Tabbed act selector with glowing unlock states, completion badges, and lore names
🔄
Progress Reset One-click progress wipe for replaying the full experience from scratch
